# F.R.I.D.A.Y (v1)

F.R.I.D.A.Y is a personal Node.js CLI app for managing daily tasks and habits in a clean, focused terminal workflow.

This project is still in development, and this is **Version 1**.

## Why I built this

I wanted a lightweight daily system that feels human and simple:
- track tasks
- build habits
- keep streaks visible
- stay in terminal without distractions

## Current Features (v1)

- Add tasks and habits
- Mark items done or skipped
- Habit streak tracking
- Daily reset behavior for habits
- CLI dashboard-style welcome panel
- Helpful command suggestions for typos

## Commands

- `/add <title>` -> add a task
- `/add <title> habit` -> add a habit
- `/done <title>` -> mark done
- `/skip <title>` -> mark skipped
- `/list` -> show tasks and stats
- `/streak` -> show streak bars
- `/clear` -> remove completed tasks
- `/help` -> show help
- `/exit` -> quit

You can also type commands without `/` (for example: `list`).

## Run Locally

```bash
npm install
npm start
```
